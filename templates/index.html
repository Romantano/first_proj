Добро пожаловать на мой первый проект мини web приложения.

Краткие сведения о проекте:

Данный проект предназначен для демонстрации уровня знакомства с фреймворком Django:)

Для реализации проекта использовался следующий стек технологий: Python, Django, немного DRF, и самая малость HTML и CSS :)

В качестве IDE использовался PyCharm Edu

Фронтенд (если данное слово применимо к данному проету:)) выполнен средствами шаблонизатора Django

Ориентировочные затраты на разработку составили - 18 человеко-часов (работа выполнялась в отпуске по вечерам)

Зачем нужно приложение (идея):

Приложение призвано помочь вести учет бюджета в указанный период путем указания доходов и расходов. Приложение можно использовать для планирования месячного бюджета. Возможные улучшения функциональности: добавить функционал регистрации/входа/выхода пользователя (просто не успел),реализовать качественный интерфейс, добавить отчёты, визуализацию отчетов, добавить возможность выгрузки данных в форматы EXCEL и PDF и мн. другое.

Реализованный функционал (кратко):

  • Создание периодов
  • Создание записей внутри периода
  • Редактирование записей внутри периода
  • Удаление записей внутри периода
